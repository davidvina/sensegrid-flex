    list-hor

    @mixin list-hor($align: 'left') { ... }

    Description

    Converts default lists in a horizontal lists

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $align

    text-align values: left, right, justify

    		String'left'

    Example

    .menu{
  @include list-hor(left);
}
    .menu {
  text-align: left;
  padding: 0;
  font-size: 0;
}
.menu li {
  display: inline-block;
  list-style: none;
  margin: 0;
}
.menu li a {
  display: inline-block;
  font-size: 16px;
 }

    col

    @mixin col($grid: $grid-default, $xl: $grid, $l: $grid, $m: $grid, $s: $grid, $xs: $grid) { ... }

    Description

    Mixin que controla ancho de las columnas seguns la medida de la ventana del navegador.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $grid

    el número de columnas que tendra la columna

    		Number$grid-default
    $xl

    el ancho en columnas que tendra $screen-xl

    		Number$grid
    $l

    el ancho en columnas que tendra $screen-l

    		Number$grid
    $m

    el ancho en columnas que tendra $screen-m

    		Number$grid
    $s

    el ancho en columnas que tendra $screen-s

    		Number$grid
    $xs

    el ancho en columnas que tendra $screen-xs

    		Number$grid

    Example

      .sidebar{
   @include col($m:3);
 }
si no se incluye el parametro el valor de ancho sera 100%
